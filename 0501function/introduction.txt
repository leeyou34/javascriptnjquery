chapter 5

함수

기존에는 데이터를 저장할 때 변수를 선언하여 저장했다.
변수에는 데이터(리터럴)을 저장할 수 있지만
반면 코드는 저장할 수 없다.


변수:
1. 1개의 데이터만 저잔하다.
2. var라는 키워드를 이용하여 선언한다.
3. 문자형, 숫자형, 노리형 데이터를 보관한다.
4. 객체를 참조한다. 


함수:

1. 자바스크립트 코드를 저장한다.
2. function이라는 키워드를 이용하여 선언
3. 출력문, 제어문 등의 코드를 저장하고 데이터를 반환한다.


기본 함수 정의문은

function 함수명() {
    자바스크립트 코드;
}

참조 변수 = function() {
    자바스크립트 코드;
}

함수명();
또는 참조 변수();

일반 함수 정의 방식과 익명 함수 선언 참조 방식의 차이점

일반 함수 정의는 함수 호출 시 호이스팅 기술을 지원한다.
하지만 익명 함수 선언 참조방식은 그렇지 않다.
